About me

I'm a generalist with over 20 years of experience in the software industry. I have worked for companies of all sizes, from small startups to large corporations.

My technology stack of choice is Python or JavaScript for the backend, and React for the frontend. PostgreSQL is my preferred database. I have experience with Django, Flask, Express, and DRF for backend development. For frontend, I work with React, Preact, or vanilla JS. I prefer TypeScript for all my JavaScript projects.

I am proficient in Linux and comfortable with the command line. My daily OS is either macOS or Arch Linux. I prefer using Docker and Nix. I have been hosting my own services on a $5 VPS for years.

Microcontrollers and embedded systems are my hobby. I once built an installation for the Riga City Festival that used Arduino, a custom LED driver, and a backend built with Flask and Redis to control the lights. People could tweet their messages or enter them into a local terminal, and the messages would be flashed using Morse code.

Lately, I have been learning Rust and OCaml. This website runs on a Rust backend with my own implementation of Server Side Includes. The frontend is vanilla JS with no frameworks. The CSS is Sakura.css with a few customizations.

I speak English, Latvian, and Russian. In the past, I lived in the Netherlands and can understand some Dutch.

I have worked as an individual contributor and have been an engineering manager as well. I enjoy programming and building things more than managing people.
